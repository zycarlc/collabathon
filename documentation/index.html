Installation

Follow the steps below to setup your site template:

Setup Development: (For Local host)
  1. NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

    You'll need to install Node.js >=v18.0+ (Recommended Version)(NPM comes along with it) and serve (a NPM package)

  2. Let's see how the setup works:
    • Use terminal and navigate to the project root.
    • Then Run :npm install
    • Then Run :npm run start
    • Now in the browser go http://localhost:3000/ The app will automatically reload if you change any of the source files.
Build the project:

Once you are complete edit and are ready to publish it, first you have to make a build of the website.

You can build it with the following command:
  • Please Run:npm run build This command will create an optimized production build for our project.
  • Then, You will find build folder in your project root folder after complete process.
  • Then, You can upload all files from build folder in your server for live site.

Changing Fonts

Callum template use fonts: Poppins from the Google Fonts Library. You can change the fonts from page head in HTML files (as following code).

From the template options go to public > index.html 

<!-- Web Fonts
============================================= -->
<link rel='stylesheet' href='https://fonts.googleapis.com/css2?family=Poppins:ital,wght@0,100;0,200;0,300;0,400;0,500;0,600;0,700;0,800;0,900;1,100;1,200;1,300;1,400;1,500;1,600;1,700;1,800;1,900&display=swap' >
  • In order to change the Fonts, you will need to Edit the Above Links with your Chosen Google Fonts.
  • Then Replace the body-font variable value in the src/sass/_variables.scss File with your Newly Added Fonts.

Color Schemes

You can change your Website's Color Scheme in an instant. We have build 7 Color css file for change template primary color.

  • color-cyan.css - Cyan Color
  • color-green-yellow.css - Green Yellow Color
  • color-lime-punch.css - Lime Punch Color
  • color-orange.css - Orange Color
  • color-pale-golden-rod.css - Pale Golden Rod Color
  • color-spring-green.css - Spring Green Color
  • color-violet.css - Violet Color

Simply import Color SCSS file it in the file src/index.js
Make sure you import the color stylesheet sass/color-cyan.scss after the sass/stylesheet.scss default stylesheet linking from above mentioned 7 color css file.

Sass

We have added SASS .scss files in template. If you know how to use SASS you can change sass files and compile the css as well. You can find sass file here - src/sass

Open the src/sass/_variables.scss and Edit the values according to your needs. If you need more Advanced Setup then you can Edit the Respective Files yourself which have been branched inside the same Folder. It is completely at your discretion only to include the Required .scss Files you need to minimize the amount of CSS & including only the Styles of the Blocks you need. This can be setup in your stylesheet.scss File.

Home

Default, Image or Video background (Hero Section)

You can set default (Solid background Color) or image background or video background in hero section.

  • Step 1

    For that set appliedIntro at src/config/commonConfig.js file.

    • Default (Solid background Color):
      appliedIntro: introBackgroundConfig.default, will set default Intro.
    • Image background:
      appliedIntro: introBackgroundConfig.image, will set image background Intro.
    • Video background:
      appliedIntro: introBackgroundConfig.video, will set video background Intro.

    Example:

  • Step 2

    When you want to edit the content in Home (Intro) section. then,
    You will need to edit its style intro file to have the intro style you activated as in step 1 as well as the header and menu styles you activated.

    You can find Home (Intro) component file for each style as below:

    • Full Screen Menu:
      • Default: src/components/themes/fullScreen/IntroDefault.jsx
      • Image background: src/components/themes/fullScreen/IntroImageBg.jsx
      • Video background: src/components/themes/fullScreen/IntroVideoBg.jsx
    • Standard Menu:
      • Default: src/components/themes/StandardMenu/IntroDefault.jsx
      • Image background: src/components/themes/StandardMenu/IntroImageBg.jsx
      • Video background: src/components/themes/StandardMenu/IntroVideoBg.jsx
    • Bottom Menu:
      • Default: src/components/themes/bottomHeader/IntroDefault.jsx
      • Image background: src/components/themes/bottomHeader/IntroImageBg.jsx
      • Video background: src/components/themes/bottomHeader/IntroVideoBg.jsx
